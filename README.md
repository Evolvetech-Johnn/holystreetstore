# Holy Street Store - Monorepo

## Structure
- `frontend/`: React + Vite + Tailwind CSS application.
- `backend/`: Node.js + Express API.
- `legacy-static/`: Archived static version of the site.

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   npm run install:all
   ```
2. Start development servers:
   ```bash
   npm run dev
   ```

## Standards
- Follows international programming standards (folder structure, naming conventions).
- Backend and Frontend separation for scalability.
